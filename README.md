# Todo application with Xamarin.Forms

+ [PrismLibrary](https://prismlibrary.com)
    + MVVM
    + Behaviors
    + Dialogs
    + Navigation
+ EntityFrameworkCore
    + Sqlite data storage
    + CodeFirst
+ [MicroCharts](https://github.com/dotnet-ad/Microcharts)
    + Donut Chart
